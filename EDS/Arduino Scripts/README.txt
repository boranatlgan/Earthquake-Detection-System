
*EDS-Edited contains an .ino file which can be uploaded into Arduino UNO for collecting and sending data via ESP8266 module. The collected data can be viewed through Thingspeak website and "Earthquake Detection System" app.

*EDS-EarthquakeDetectionSystem contains another .ino with different line of scripts. The file can be uploaded into Arduino UNO for collecting data. The collected data, then can be viewed on Processing app.
